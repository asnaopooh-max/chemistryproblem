# 今後の拡張案

このアプリは基本的なランダム出題機能に対応していますが、今後は次のような機能を追加できます。

- 難易度指定機能の UI を追加し、選択した難易度のみ出題する。
- 間違えた問題だけを再出題するモード。
- グラフや表を使った問題をサポートするための画像表示機能。
- 画像問題や図解問題の追加対応。
- 単元別到達度表示や復習履歴を画面上で確認する機能。
- PWA 化してオフラインでの起動をサポートする。
- 教員用の問題編集画面を追加し、問題データを直接編集できる機能。
- 解説の表示形式を「詳しく表示 / 簡単に表示」で切り替える機能。
- 正解率や出題済み問題数などを視覚的にグラフで表示する機能。
- CSV や JSON 形式で問題データをインポート・エクスポートする機能。
